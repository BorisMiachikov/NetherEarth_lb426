# Редактор уровней — Nether Earth LB426

> Встроенный редактор для создания и редактирования карт игры.  
> Статус: ✅ Полностью реализован (фаза 11 завершена).

---

## 1. Запуск редактора

В **главном меню** → кнопка «Редактор уровней».  
Для выхода из редактора нажмите `ESC` → возврат в главное меню.

> Если карта была изменена (`dirty=true`), при выходе появится диалог с кнопками «Сохранить», «Не сохранять», «Отмена».

---

## 2. Управление камерой

| Клавиша | Действие |
|---------|----------|
| `W/S` | Камера вперёд/назад |
| `A/D` | Камера влево/вправо |
| `Z` | Вращение камеры влево |
| `C` | Вращение камеры вправо |
| `Scroll` | Зум (приближение/отдаление) |

Камера позиционируется по центру карты при входе в редактор.

---

## 3. Интерфейс

### Левая панель — Панель инструментов

**Выбор инструмента:**
- **Terrain Brush** — рисование типов клеток
- **Place Factory** — размещение фабрики
- **Place Warbase** — размещение базы
- **Place Player Spawn** — точка появления скаута
- **Erase** — удаление структуры или сброс клетки в Open

**Для Terrain Brush:**
- Тип клетки: Open / Rock / Pit / Sand
- Размер кисти: 1×1 / 3×3 / 5×5

**Для Place Factory:**
- Тип фабрики: Chassis / Cannon / Missile / Phasers / Electronics / Nuclear
- Команда (владелец): Player / Enemy / Neutral

**Для Place Warbase:**
- Команда: Player / Enemy

### Правая панель — Свойства карты

- **Имя карты** — используется как имя файла
- **Описание** — отображается при выборе сценария
- **Начальные ресурсы** (опционально) — переопределяют стартовые запасы игрока при загрузке сценария. Чекбокс включает секцию из 7 полей (General/Chassis/Cannon/Missile/Phasers/Electronics/Nuclear). Если выключено — в сценарии поле `initial_resources: None` и в игре применяются defaults (`PlayerResources::with_starting_values`).
- **Размер**: 32×32 / 64×64 / 96×96 *(изменение размера сбрасывает карту)*
- **Статистика**: количество фабрик по типам, варбейсов, клеток по типам

---

## 4. Рисование клеток

1. Выбрать **Terrain Brush** в панели инструментов
2. Выбрать тип клетки (Open/Rock/Pit/Sand)
3. Выбрать размер кисти
4. **ЛКМ** на клетке — применить тип

Клетки со структурами нельзя перекрасить — сначала удалите структуру инструментом **Erase**.

**Визуальный отклик:**
- Сетка карты отображается тонкими белыми линиями (gizmos, alpha=0.08)
- Координаты клетки под курсором показаны в левой панели
- Клетки перестраиваются мгновенно (observer `on_rebuild_terrain_cell`)

---

## 5. Размещение структур

### Фабрики и варбейсы

1. Выбрать **Place Factory** или **Place Warbase**
2. Задать тип и команду в левой панели
3. **ЛКМ** на клетке — разместить структуру

**Ограничения:**
- Клетка должна быть **Open** (не Rock/Pit и без существующей структуры)
- Нельзя размещать две структуры в одной клетке

**Предпросмотр (gizmos):**
- Фабрики: белый квадрат на полу + вертикальная линия-«флаг» цвета команды
- Варбейсы: двойной квадрат + крест цвета команды
- Точка спавна: жёлтый ромб + крест

Цвета команд:
- 🟢 Player — зелёный (0.15, 0.85, 0.25)
- 🔴 Enemy — красный (0.95, 0.15, 0.15)
- ⚪ Neutral — серый (0.75, 0.75, 0.75)

### Точка спавна игрока

1. Выбрать **Place Player Spawn**
2. **ЛКМ** на клетке — установить точку появления скаута

На карте может быть **только одна** точка спавна игрока.

---

## 6. Удаление

1. Выбрать **Erase**
2. **ЛКМ** на клетке со структурой — удалить структуру
3. **ЛКМ** на клетке terrain — сбросить в Open

---

## 7. Сохранение карты

Кнопка **«Сохранить»** в левой панели.

**Валидация перед сохранением:**
- Минимум 1 Player Warbase
- Минимум 1 Enemy Warbase
- Валидная точка спавна игрока
- Непустое имя карты

При ошибке — сообщение в egui-диалоге, сохранение не происходит.

**Результат:**
- `data/maps/{name}.ron` — данные карты (сетка + структуры)
- `data/scenarios/{name}.ron` — сценарий (имя, описание, ссылка на карту)

Сохранённый сценарий **автоматически появляется** в списке выбора сценария главного меню.

### Формат maps/*.ron

```ron
(
    width: 64,
    height: 64,
    cells: [ /* 64×64 массив CellType */ ],
    factories: [
        (x: 8, y: 8, factory_type: Chassis, team: Player),
        // ...
    ],
    warbases: [
        (x: 8, y: 8, team: Player),
        (x: 56, y: 56, team: Enemy),
    ],
    player_spawn: (8, 8),
)
```

---

## 8. Загрузка карты

Кнопка **«Открыть»** в левой панели.

Отображается список файлов из `data/maps/`. Клик по имени — загружает карту в редактор.

---

## 9. Новая карта

Кнопка **«Новая карта»** в левой панели.

- Все клетки сброшены в Open
- Все структуры удалены
- Имя сброшено в «new_map»
- Точка спавна — (0, 0)

## 11. Советы по созданию карт

- **Симметрия** — обе стороны должны иметь равный доступ к ресурсам для честного баланса
- **Распределение фабрик** — по 2 фабрики каждого типа (12 всего) — стандарт карты `default.ron`
- **Sand** — добавляет стратегическую глубину: замедляет Wheels/Bipod, создаёт зоны преимущества для AntiGrav
- **Rock/Pit** — используйте для создания коридоров и тактических позиций
- **Точка спавна** — ставьте рядом с Player Warbase, но не вплотную
- **Тестируйте** загрузку карты через «Новая игра» → выбор сценария сразу после сохранения
