# Механики игры — Nether Earth LB426

> Руководство по всем игровым механикам.

---

## 1. Концепция

Игрок управляет **Скаутом** — летающим командным модулем. Скаут сам по себе не боевая единица; его задача — **строить роботов** и **отдавать им приказы**.

Роботы автономно воюют: захватывают фабрики, добывают ресурсы, атакуют противника. Финальная цель — **уничтожить вражескую базу (Warbase)** с помощью ядерного заряда.

---

## 2. Скаут (командный модуль)

### Управление

| Клавиша | Действие |
|---------|----------|
| `W/A/S/D` | Движение (относительно угла камеры) |
| `Q/E` | Вверх / вниз |
| `Scroll` | Зум камеры |
| `Z/C` | Вращение камеры орбитально |
| `B` | Открыть Builder UI (рядом с варбейсом) |
| `ESC` | Пауза |
| `F1` | Справка по клавишам |

Скаут **ограничен границами карты** и не может пролететь сквозь структуры.

---

## 3. Роботы

### Модульная система

Каждый робот собирается из модулей:

#### Шасси (обязательно — 1 слот)

| Тип | HP | Скорость | Мобильность | Стоимость | Особенности |
|-----|----|---------:|-------------|-----------|-------------|
| Wheels | 50 | 6.0 | 0.6 | Chassis×15 + 5G | — |
| Bipod | 40 | 7.5 | 0.8 | Chassis×20 + 10G | — |
| Tracks | 80 | 5.0 | 0.9 | Chassis×40 + 15G | Лучшая броня |
| AntiGrav | 30 | 10.0 | 1.0 | Chassis×60 + 25G | **Летает** над любым terrain |

`AntiGrav` игнорирует Rock/Pit/Sand — идеален для разведки и ядерных ударов.

#### Оружие (1–3 слота, минимум 1)

| Тип | Урон | Дальность | Перезарядка | Стоимость | Тип атаки |
|-----|-----:|----------:|-------------|-----------|-----------|
| Cannon | 15 | 10 | 1.2 с | Cannon×10 + 5G | Hitscan (мгновенно) |
| Missile | 45 | 30 | 3.0 с | Missile×25 + 10G | Projectile (самонаведение) |
| Phasers | 8/тик | 6 | 0.15 с | Phasers×30 + 15G | Hitscan (непрерывный) |

- **Cannon** — надёжный универсал
- **Missile** — максимальный урон и дальность, медленный старт
- **Phasers** — высокий DPS в ближнем бою (4× DPS от Cannon)

#### Электроника (опционально)

Стоимость: Electronics×20 + 10G

| Бонус | Значение |
|-------|----------|
| Точность | +30% |
| Скорострельность | +20% |
| Радиус радара | 8 → **20** единиц |
| Время захвата | −30% (10 с → **7 с**) |

Электроника критична для захватчиков и снайперов.

#### Ядерный заряд (опционально)

Стоимость: Nuclear×50 + 30G

- Радиус взрыва: **8 единиц**
- Задержка детонации: **2 секунды** (робот активирует при достижении цели)
- Уничтожает **всё** в радиусе: роботы, фабрики, варбейсы
- Уничтожает и **самого носителя**
- **Единственный** способ уничтожить Warbase

### Строительство роботов

1. Подлететь к своему **Warbase**
2. Нажать `B` → открывается Builder UI
3. Выбрать шасси, оружие (до 3 слотов), опциональные модули
4. Проверить стоимость (красный = нехватка ресурсов)
5. Нажать «Построить» → робот встаёт в очередь производства
6. По завершении → спавн у входа в Warbase

Время постройки: `sum(стоимости модулей) × 0.5` секунд.

## 4. Система приказов

### Выбор роботов

| Клавиша | Действие |
|---------|----------|
| `ЛКМ` на роботе | Выбрать одного |
| `Shift+ЛКМ` | Добавить/убрать из выделения |
| `Ctrl+1-9` | Сохранить группу |
| `1-9` | Вызвать группу |
| Кнопка «✕» в панели | Снять выделение |

Выбранный робот подсвечивается жёлтым кругом.

### Типы приказов

| Приказ | Способ выдачи | Описание |
|--------|--------------|----------|
| **MoveTo** | `ПКМ` на земле | Двигаться в точку |
| **SeekAndDestroy** | Кнопка в UI | Найти и атаковать ближайшего врага |
| **SeekAndCapture** | Кнопка в UI | Найти и захватить ближайшую нейтральную/вражескую фабрику |
| **DestroyEnemyBase** | Автоматически (ядерный робот) | Идти к Warbase врага, взорваться |
| **Defend** | Кнопка в UI | Охранять текущую позицию |
| **Patrol** | `P+ПКМ` (накопление точек) | Патрулировать по маршруту |
| **Idle** | Кнопка «Стоп» | Стоять на месте |

Роботы без электроники видят врагов в радиусе **8 единиц**; с электроникой — **20 единиц**.  
При отсутствии видимых целей — исследуют карту самостоятельно.

## 5. Экономика

### Ресурсы

| Ресурс | Источник производства | Применение |
|--------|----------------------|------------|
| General | Warbase (+5/день), каждая фабрика (+2/день) | Любые постройки |
| Chassis | Chassis-фабрика (+5/день) | Шасси роботов |
| Cannon | Cannon-фабрика (+5/день) | Орудия |
| Missile | Missile-фабрика (+5/день) | Ракеты |
| Phasers | Phasers-фабрика (+5/день) | Фазеры |
| Electronics | Electronics-фабрика (+5/день) | Электроника |
| Nuclear | Nuclear-фабрика (+5/день) | Ядерные заряды |

**Стартовые ресурсы:** General:50, Chassis:20, Cannon:15, Missile:10, Phasers:10, Electronics:10, Nuclear:5

### Игровой день

1 игровой день = **30 реальных секунд** (настраивается в `configs/game.ron`).  
Производство начисляется в конце каждого дня. HUD показывает текущий запас и прирост.

### Фабрики

На карте 12 нейтральных фабрик (по 2 каждого специфического типа). Нейтральные — не производят ресурсы до захвата.

**Приоритет захвата:** Chassis и Electronics — наиболее универсальны. Nuclear — критична для победы.

---

## 6. Механика захвата

1. Направить роботу приказ **SeekAndCapture**
2. Робот находит ближайшую нейтральную/вражескую фабрику
3. Подходит к зданию и начинает захват
4. Прогресс-бар виден над структурой
5. По завершении — здание меняет цвет команды

| Условие | Время |
|---------|-------|
| Без электроники | 10 секунд |
| С электроникой | 7 секунд |
| Захват вражеской (сначала → Нейтральная) | ×2 (20 → нейтральная, +10 → своя) |

Если во время захвата робот уходит — **прогресс сбрасывается**.  
Если прибывает вражеский робот — прогресс уменьшается (перетягивание).

## 7. Карта

Типы клеток:

| Тип | Проходимость | Эффект |
|-----|-------------|--------|
| **Open** | ✅ Все | Нет |
| **Rock** | ❌ Заблокировано (кроме AntiGrav) | — |
| **Pit** | ❌ Заблокировано (кроме AntiGrav) | — |
| **Sand** | ✅ С замедлением | -30% скорости для Wheels/Bipod |
| **Structure** | ❌ Клетка занята зданием | — |

Размер карты: **64×64** клеток (1 клетка = 1 игровая единица).

---

## 8. AI-противник

ИИ использует **Utility scoring** — каждое действие оценивается по полезности:

- При малом количестве фабрик → приоритет захвата
- При угрозе (враги рядом) → приоритет защиты
- При достаточном запасе Nuclear → строит ядерного робота → **DestroyEnemyBase**

ИИ работает **по той же экономике**, что и игрок — без читов.

Интервал принятия решений: настраивается в `configs/ai.ron` (параметр `decision_interval`).

---

## 9. Победа и поражение

**Победа:** все варбейсы противника уничтожены ядерным взрывом.  
**Поражение:** все ваши варбейсы уничтожены.

Экран Game Over показывает: количество прошедших дней, количество захваченных фабрик обеих сторон.

---

## 10. Сохранения

- **Автосохранение**: автоматически каждый игровой день (`saves/autosave.ron`)
- **Ручные слоты**: 3 именованных слота через меню паузы (Сохранить/Загрузить)
- Формат: RON (человекочитаемый текст)
- Кнопка «Продолжить» в главном меню загружает автосохранение

---

## 11. Пауза

`ESC` во время игры → меню паузы:
- **Продолжить** — вернуться в игру
- **Сохранить / Загрузить** — слоты сохранения
- **Главное меню** — выход с подтверждением (если игра не сохранена)
- **Выход** — закрыть приложение

Вся игровая логика (FixedUpdate) заморожена на паузе. UI отвечает.
