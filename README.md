> Современное переосмысление классической RTS **Nether Earth** (1987).  
> Изометрическая 3D стратегия на Rust + Bevy 0.18.

[![Rust](https://img.shields.io/badge/Rust-stable-orange)](https://www.rust-lang.org/)
[![Bevy](https://img.shields.io/badge/Bevy-0.18-blue)](https://bevyengine.org/)
[![Status](https://img.shields.io/badge/Статус-Alpha-yellow)](ROADMAP.md)

## О игре

Игрок управляет **летающим командным модулем (Скаутом)** — единственной боевой единицей под прямым контролем. Скаут строит кастомизируемых роботов, которые автономно воюют за **фабрики ресурсов**. Цель — уничтожить вражескую базу ядерным зарядом.

Противник управляется **ИИ-командиром** с Utility-based решениями — такой же экономикой, без читов.

## Сборка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/BorisMiachikov/NetherEarth_lb426.git
cd NetherEarth_lb426

# Запуск в режиме разработки (быстрая компиляция)
cargo run --features dev

# Запуск с отладочными инструментами
cargo run --features dev,debug_tools

# Релизная сборка
cargo build --release
```

**Требования:** Rust Stable, Windows / Linux / macOS.

## Управление

### Скаут (командный модуль)

| Клавиша | Действие |
|---------|----------|
| `W/S/A/D` | Движение вперёд/назад/влево/вправо |
| `Q / E` | Движение вверх / вниз |
| `Scroll` | Зум камеры |
| `Z / C` | Вращение камеры влево/вправо |
| `B` | Открыть меню строительства (рядом с варбейсом) |
| `ESC` | Пауза |
| `F1` | Справка по клавишам |

### Роботы

| Клавиша | Действие |
|---------|----------|
| `ЛКМ` на роботе | Выбрать |
| `Shift+ЛКМ` | Добавить к выделению |
| `Ctrl+1-9` | Сохранить группу |
| `1-9` | Выбрать группу |
| `ПКМ` на земле | Команда MoveTo |
| `P+ПКМ` | Добавить точку патрулирования |
| Панель команд | SeekAndDestroy / SeekAndCapture / Defend / Idle |

### Редактор уровней

| Клавиша | Действие |
|---------|----------|
| `W/S/A/D` | Перемещение камеры |
| `Z / C` | Вращение камеры |
| `Scroll` | Зум |
| `ЛКМ` | Применить инструмент |
| `ESC` | Выйти в главное меню |

## Игровые механики

### Модульная система роботов

Каждый робот собирается из компонентов:
- **Шасси** (обязательно): Wheels / Bipod / Tracks / AntiGrav
- **Оружие** (1-3 слота): Cannon / Missile / Phasers
- **Электроника** (опц.): +точность, +скорострельность, +радар
- **Ядерный заряд** (опц.): взрыв радиус 8 ед., единственный способ уничтожить варбейс

### Экономика

7 типов ресурсов: General, Chassis, Cannon, Missile, Phasers, Electronics, Nuclear.  
Каждая захваченная фабрика производит +5 специфического + 2 General за игровой день (30 секунд реального времени).

### Победа / Поражение

- **Победа:** уничтожить все варбейсы противника
- **Поражение:** все ваши варбейсы уничтожены
- Варбейсы неуязвимы к обычному оружию — нужен ядерный заряд

## Data-driven дизайн

Весь баланс игры хранится в RON-файлах и может быть изменён **без пересборки**:

```
configs/
├── chassis.ron       — HP, скорость, стоимость шасси
├── weapons.ron       — урон, дальность, перезарядка
├── electronics.ron   — бонусы электроники
├── nuclear.ron       — радиус взрыва, задержка
├── factories.ron     — скорость производства
├── ai.ron            — агрессивность, интервалы решений
└── game.ron          — скорость времени, размер карты
```
